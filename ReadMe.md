### @Column

칼럼의 이름을 이용하여 지정된 필드나 속성을 데이블의 칼럼에 매핑한다.
생략되면 속성과 같은 이름의 칼럼으로 매핑된다.

> @Column(name = "bo_num", length = 10)

private Long num; 

일경우 bo_num 컬럼으로 매핑
